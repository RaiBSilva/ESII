	REGRAS DE NEGÓCIOS

	- Os dados do site da transparência da prefeitura de Mogi das Cruzes  precisam ser atualizados pelo menos a cada 30 dias de acordo com a lei da transparência; //Fonte: Renato M.
	- A prefeitura de Mogi das Cruzes deve disponibilizar os dados referentes às despesas, receitas, fornecedores, programas, ações e projetos para o público, de acordo com a lei federal ei n° 6.924,de 25 de julho de 2009; //Fonte: Renato M.
		-Esses dados devem conter informações sobre os servidores públicos, obras públicas e concursos públicos;
	- Cabe ao Ministério Público garantir a veracidade dos dados informados por meio de fiscalizações;

	REQUISITOS

	LEGENDA: (-) NECESSÁRIO
		 (+) SE SOBRAR TEMPO
		 (*) A DECIDIR

	- O sistema deve, a cada 24H, atualizar seu banco de dados com os dados proveniente dos sites "http://www.transparencia.pmmc.com.br/", "http://www.cmmc.com.br/"; //Fonte: Renato M.
	- O sistema deve ter uma interface intuitiva, responsiva para dispositivos móveis e limpa; //Fonte: Rai B.
	- O sistema deve fornecer para o usuário os seguintes dados sobre cada servidor pública: //Fonte: Rai B.
		- Nome;
		- Cargo;
		- Regime;
		- Referência;
		- Total Bruto do salário (em R$).
		- Total Líquido do salário (em R$).
		- Total de Descontos (em R$).
		- Cada remuneração e seu valor (em R$).
		- Cada desconto e seu valor (em R$).
	- O sistema deve fornecer a possibilidade de buscar e filtrar os dados dos servidores por qualquer um dos tipos de dados armazenados; não somente isso, o sistema deve fornecer os filtros:
		- De intervalo de salário;
		- De intervalo de descontos;
		- De intervalo de referência;
	- O sistema deve possibilitar a comparação dos salários de vários servidores públicos; //Fonte: Rai B.
	- O sistema deve mostrar a assiduidade dos parlamentares, mostrando:
		- Quantas faltas cada parlamentar tem;
		- Os dias nos quais o parlamentar faltou;
		- O motivo de cada falta;
	- O sistema deve apresentar os dados das diárias e passagens dos servidores públicos, em específico: //Fonte: Rai B.
		- Servidor que fez a solicitação de diária;
		- Período;
		- Data e horário de saída e chegada;
		- Destino / Motivo;
		- Valor (em R$);
	- O sistema deve fornecer a possibilidade de buscar e filtrar os dados das diárias por qualquer um dos tipos de dados armazenados; além diso, o sistema deve oferecer filtros: //Fonte Rai B.
		- De intervalo de data;
		- De intervalo de custo;
	- O sistema deve apresentar os dados dos projetos de leis propostos na câmara municipal de Mogi das Cruzes, em específico: //Rai B.
		Nº;
		Autor;
		Assunto;
		Anotação;
	- O sistema deve fornecer a possibilidade de buscar e filtrar os dados dos projetos de lei por qualquer um dos tipos de dados armazenados; além diso, o sistema deve oferecer filtros: //Rai B.
		- De tipo de Lei;
		- De ano;
